# Local Web Chain 001

Sample benchmark case placeholder for validating the PenHub comparison harness.

This directory intentionally contains no exploitable target image yet. The sample fixtures in `harness/penhub/fixtures`
are marked with `isSampleData: true` and exist only to exercise comparison, report generation, and UI rendering paths.

Fair-run constraints for a future controlled run:

- same model
- same target
- same max wall time
- same token ceiling
- same tool allowlist
- same number of attempts
